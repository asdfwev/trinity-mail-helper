Trinity 메일 도우미

Privacy Policy (개인정보 처리방침)
최종 수정일: 2026년 4월 16일

1. 개요

본 문서는 Chrome 확장 프로그램 "Trinity 메일 도우미"(이하 "본 확장")의 개인정보 수집 및 처리 방침을 설명합니다. 본 확장은 그룹웨어 메일 발송 전 확인창을 표시하여 사용자의 실수를 방지하는 생산성 도구입니다.

2. 수집하는 정보

본 확장은 사용자의 어떠한 개인정보도 수집, 저장, 전송하지 않습니다.

다음 정보들은 오직 사용자의 로컬 브라우저 내부에서만 처리됩니다:

  • 메일 작성 화면의 본문 텍스트 — 첨부파일 키워드 감지 목적으로만 읽으며, 외부로 전송되지 않습니다.
  • 첨부파일 개수 — 리마인더 판정 목적으로만 사용되며, 파일 내용이나 파일명은 수집하지 않습니다.
  • 사용자 설정값 (확인 대기 시간, 체크 문구, 감지 키워드 등) — Chrome의 chrome.storage.local API를 통해 사용자 본인의 브라우저에만 저장됩니다.

3. 외부 서버로의 데이터 전송

본 확장은 기본적으로 어떠한 외부 서버로도 데이터를 전송하지 않습니다.

단, 사용자가 선택적으로 활성화할 수 있는 AI 메일 요약 기능을 사용할 경우에 한하여, 해당 메일의 본문 내용이 Anthropic Claude API(api.anthropic.com)로 전송됩니다. 이 기능은 사용자가 자신의 API 키를 직접 입력하고 명시적으로 "요약" 버튼을 클릭했을 때만 동작하며, Anthropic의 개인정보 처리방침이 적용됩니다.

4. 권한 사용 목적

  • storage — 사용자 설정을 브라우저 로컬에 저장하기 위함
  • host_permissions (api.anthropic.com) — AI 요약 기능 사용 시 Claude API 호출을 위함 (선택 기능)
  • content_scripts (sonotrinitygroup.com) — 그룹웨어 페이지에서 확인창을 표시하기 위함

5. 제3자 제공

본 확장은 사용자 정보를 어떠한 제3자에게도 제공, 판매, 공유하지 않습니다.

6. 광고 및 추적

본 확장은 광고를 표시하지 않으며, 사용자의 행동을 추적하지 않습니다. Google Analytics 등 분석 도구도 사용하지 않습니다.

7. 데이터 삭제

본 확장을 Chrome에서 제거하면, 저장된 모든 설정값이 자동으로 삭제됩니다. 별도의 삭제 요청 절차는 필요하지 않습니다.

8. 변경 사항

본 방침은 예고 없이 변경될 수 있으며, 중요한 변경 사항은 본 페이지 상단의 "최종 수정일"을 통해 고지됩니다.

9. 문의
